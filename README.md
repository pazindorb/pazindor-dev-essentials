This module was created to facilitate the work of developers and macro creators. It provides useful features:
- InputDialog - allow users to send questions to other users and wait for the response.
- TokenSelector - allow user to select tokens. Returns selected tokens.
- TextEditor - Prose Mirror based text editor.
- BaseDialog - ApplicationV2 dialog with listeners that modify dialog values, can also update database objects (such as an actor or item).
- utils - a collection of useful methods that can significantly improve the development process.
